
# Getting Started

Clone the project : 
```
git clone https://github.com/RadwanHegazy/Todo-RadwanMVT
```

open the donwloaded project : 
```
cd Todo-RadwanMVT
```

install require packages : 
```
pip install -r requirments.txt
```


### Run Server 
```
python3 server.py
```

### Now go to your browser and open : http://localhost:8080
